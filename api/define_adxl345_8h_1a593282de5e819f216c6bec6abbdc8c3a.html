ADXL345_FIFO_ENTRIES

32 * (X, Y, Z); 2 bytes each coordinate

ADXL345 watermark configuration.

Sensor provides 32 FiFo entries 2-bytes each. Recommended watermark level is about 75% of FiFo. Setting the watermark level too high might leave too little room for buffering until respective routine clears the FiFo. This might increases the probability of FiFo overflow (which is asserted anyway).
